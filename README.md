Python Beginner Programs

This repository contains two simple Python programs written as part of learning programming fundamentals.
These programs focus on input handling, basic operations, validation, and formatted output.

1. Calculator Program
Description

This program takes two integer values from the user and performs:

Addition

Subtraction

Multiplication

Division (with division by zero handling)

It also validates input to prevent crashes caused by invalid data.

2. Name Greeting Program
Description

This program asks the user to enter their first and last name and displays a welcome message.
It ensures only alphabetic input is accepted.
