Python Beginner Programs

This repository contains two Python programs created to practice programming fundamentals.
The focus of these programs is on input handling, arithmetic operations, validation, and clean output formatting.

Programs Included
1. Calculator Program

Description

This program accepts two integer values from the user and performs the following operations:

Addition

Subtraction

Multiplication

Division (with division-by-zero handling)

It also includes input validation to prevent crashes caused by invalid entries.

2. Name Greeting Program

Description

This program asks the user to enter their first and last name and then displays a welcome message.

It ensures that:

Only alphabetic characters are accepted

Extra spaces are removed from input

Output is properly formatted
